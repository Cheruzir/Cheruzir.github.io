Переменные - хранение значения

Типы данных - String(Строка, символы) , Numbers(Числа - целые и дробные) , Boolean (Логичные значения: True, false)
NaN = Not a Number (Не числа - строка 1 символ, все что угодно)
Infinity = БЕСКОНЕЧНОСТЬ (Нифига себе, Я ЭТО НЕ ЗНАЛ)

ВВОД ДАННЫХ: 
1. Ввод данных
2. Обработка данных
3. Вывод результата

Ввод - команда prompt()
Вывод - console.log()

Оператор условия выбора:
Мы даем  указание компу выбрать ветку исполнения кода 

Например:
let day =  "Monday"
let x = -1
switch (day) {
    case "Monday": 
    if (x == 0){
        console.log("понедельник")
    }
    case "Tuesday": console.log("вторник")
    case "Wendsday": console.log("среда")
    case "Thursday": console.log("четверг")
    case "Friday": console.log("пятница")
    default: console.log("ERROR IN DATA, CHOOSE THE DAY")
}
